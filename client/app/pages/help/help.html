Deck import bookmarklet

  1. Drag and drop the following link Import deck to you browser's bookmark bar
  2. Open page with deck in one of supported format listed bellow
  3. Click the Import deck bookmark on your bookmark bar

Hints

  • Click on scroll image or name in right panel opens detail page (with recent and bookmarked scrolls)
  • To open scroll directly hold one of following keys Ctrl, Alt, or Option during click on a card
  • Click on unit preview triggers attack animation
  • Try Scrolls > View > All to see more information at once
  • Every trait, property in scroll detail and number in stats is linked to filter
  • Holding Ctrl, Alt, or Option keys during click on resource or cost filter sets other items to inverted state
  • Use the import bookmarklet together with setting My scrolls at Scrolls > Properties
  • All your deck data are store locally, use icon to get shareable link at deck Properties tab
  • Check additional settings under icon

User data

This web application stores all user data locally in browser using HTML5 Web storage (localStorage). Some user data like deck or user card collection can be share via link.

Privacy

This web application is using Google Analytics to collect non-personal anonymous data about user interaction with this web application like clicked links and user interface elements. Unique auto-generated numerical user identification is stored in HTML5 Web storage (localStorage).
